RU

Этот калькулятор создан для расчёта урона навыков персонажей. Он учитывает основные характеристики, тип цели и наличие талантов. Разработчики уже сами не понимают, как устроены навыки персонажей, по этому при расчётах возможна небольшая погрешность.

При вводе значений, указывайте напрямую числовое значение процентов (например 30 свирепости). В разделе талантов, в скобках, например (I) или (III) указывается номер ветки талантов, что бы не возникало путаницы.

В случае возникших вопросов обращайтесь в Discord k1ber3301

EN

This calculator was created to calculate the damage of character skills. It takes into account the main stats, type of target and the talents. The developers themselves no longer understand how character skills work, so there may be a slight error in the calculations.

When entering stats, type directly the numerical value of the percentage (for example, 30 ferocity). BUT, in the target protection input fields, type an integer value (for example, 9000 protection). In the talent section, in brackets, for example (I) or (III), the number of the talent branch, so that there is no confusion.
